# Expense Tracker

A command-line expense tracker built with Python. This project is designed to practice object-oriented programming, Git/GitHub workflow, input validation, and clean software design.

## Features

Current features:

- Add a new expense
- Store expenses in memory using a list
- Represent each expense using an `Expense` class
- Validate expense categories
- Basic menu-driven interface

Planned features:

- View all expenses
- Edit expenses
- Delete expenses
- Save and load expenses from a file
- Generate spending summaries by category and month

## Technologies Used

- Python
- Git
- GitHub
